Goal:

         Show the origins of each macrogenre   


Creating the Subdataset:

  • Checked Genre Origins within Each Macrogenre
          We first, with the help of chatGPT, analyzed the origins associated with each genre composing a given macrogenre, ensuring all contributing genres were correctly mapped to their respective origins.     

  • Aggregated Origins and Genre Counts
          After removing the original genre column, we summed all origin values and combined them with the total counts of genres belonging to each macrogenre.     

  • Converted Origin Counts to Percentages
          The absolute counts for each origin were replaced with their corresponding percentages relative to the total size of the macrogenre.     

  • Validated Geographic Coordinates
          We verified and assigned latitude and longitude values for each country to enable accurate geographic representation.     

  • Structured Final Macrogenre Columns
          Finally, we created one column per macrogenre containing only the percentage distribution of each origin within that macrogenre.     



Final Structure

Macrogenre The macrogenre
Origin The origins of each genre composing the macrogenre
Latitude The latitude of each country
Longitude The longitude of each country
All Genres The percentage of each origin composing the macrogenre
One Column per Macrogenre The percentage of each origin composing the macrogenre
