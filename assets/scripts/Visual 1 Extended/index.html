Goal

    The primary objective is to identify seasonally reliant genres, which are those that exhibit a significant spike in listening activity corresponding to a specific time of year. We also want to determine the most popular genres per season.

Creating the subdataset

    The first step involved converting the date entries into seasonal categories. Given the dataset's “YYYY-MM-DD” format, we extracted the month component from each record and used it to populate a new season column, based on standard Swiss seasonal definitions.  

    Then we transformed the rank into the variable “Value” using the following logic:  

Position 1: 100 points
Position 2: 99 points
…  

    After that, we summed the scores per season per macrogenre. Lastly, we scaled all values using a min–max scaler to show differences between seasons while avoiding distortion from genres with larger audiences, such as Pop.  

Final Structure

season The season
genre The macrogenre
value_original Not scaled rank point count
value_scaled Scaled rank point count
